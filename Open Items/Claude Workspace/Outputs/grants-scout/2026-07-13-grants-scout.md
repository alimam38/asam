# Grants Scout — 2026-07-13 (first run)

First-ever run of the grants scout. Focus: Turner Theological Seminary (AME seminary at the
Atlanta University Center / ITC; ATS + SACSCoC recent candidacy; 501(c)(3), small institution).
Purpose areas per `grants-focus.md`: theological education & faculty/program development,
capital/facilities (Talbot Hall, GLBA network), student scholarships, accreditation capacity,
thriving-in-ministry programs.

**Method note:** the "Granted" and Kindora grants-database MCP servers both returned
"tool call requires approval" in this non-interactive scheduled session, so this run is
web-search-verified only. Approving those connectors for scheduled runs would deepen future
sweeps. All deadlines below come from tool results; anything unconfirmed is marked **verify**.

**Headline:** the FY2026 NSGP window in Georgia closes **July 15 — two days from now.**

## Opportunities

### 1. DHS/FEMA — FY2026 Nonprofit Security Grant Program (via GEMA) — URGENT
- **What it funds:** Physical-security hardening for high-risk nonprofits (target hardening,
  equipment, planning, training). Ties directly to the GLBA network / facilities-security thread.
- **Fit:** Facilities & security (active grant thread — 2025 cycle already on file; this is the
  watched "next NSGP round").
- **Deadline:** Georgia SAA (GEMA) application window **June 26 – July 15, 2026** (federal NOFO
  deadline July 24, 2026, but the state deadline governs). NOFO published June 24, 2026.
- **Award:** $300M national program; per-site award cap — **verify** in the FY26 NOFO
  (historically ~$150K/site).
- **Eligibility flags:** 501(c)(3) nonprofits (religious/educational orgs qualify); must apply
  through Georgia's SAA (GEMA), located in Georgia; risk/vulnerability assessment required.
  Questions: HSgrants@gema.ga.gov.
- **Source:** https://gema.georgia.gov/nonprofit-security-grant-program ·
  https://www.fema.gov/sites/default/files/documents/fema_gdp_FY26-NSGP_NOFO.pdf ·
  https://simpler.grants.gov/opportunity/f093f16b-616d-4509-8da8-84887d36771a

- **Deadline:** Applications due **August 14, 2026**.
- **Award:** Not published — **verify** (program support rather than a fixed grant amount).

### 4. Lilly Endowment — Pathways for Tomorrow Initiative (next-round watch)
- **What it funds:** Theological schools' institutional capacity to prepare/support pastoral
  leaders. Latest round: 45 large-scale collaboration grants ($2.5M–$10M) announced Dec 2025;
  assessment/planning grants (up to $50K) and Individual School Implementation Grants run
  through the ATS coordination program.
- **Fit:** Theological education + accreditation/institutional-effectiveness capacity (the
  watched Lilly cycle).
- **Deadline:** No currently confirmed open deadline — last planning-grant deadlines were
  Dec 19, 2024 / Jan 31, 2025; implementation-grant timing for the expanded round — **verify**
  via ATS Pathways Phase 3 page (site returned 503 during this run).
- **Award:** $50K (planning) up to $2.5M–$10M (collaboration).
- **Eligibility flags:** **ATS-accredited** schools in US/Canada — verify whether candidacy
  status qualifies; coordinate with ITC where relevant.
- **Source:** https://lillyendowment.org/pathways-for-tomorrow-initiative/ ·
  https://www.ats.edu/Pathways-for-Tomorrow-Phase-3
