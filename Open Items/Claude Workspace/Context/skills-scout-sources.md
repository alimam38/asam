# Skills Scout — sources (tunable)

The daily skills-scout reads this file for WHERE to look. Edit freely — add sources you trust,
drop dead ones, re-tier as you learn. Each source carries a trust tier; the scout repeats that
trust note in the digest. Philosophy: take the best of what's available and synthesize — no single
directory is complete, so we patch several together.

## Notes & caveats
- Quick-list audit (2026-06-21): claude.com/plugins = solid (Tier 1). agentskills.io = the standard/spec
  site, not a feed (moved to reference). "DiscoverAISkills" and "Agensi" could not be verified — dropped,
  replaced by the verified directories above.
- **X/Twitter is high-signal but login-walled and JS-heavy** — an automated web scout only catches what's
  indexed or quoted elsewhere; it will NOT deeply read timelines. For real X coverage, follow a few accounts
  manually, or add an X/Twitter MCP connector later.
- Trust tiers map to the digest's trust note: Tier 1 = Anthropic-verified; Tier 2/3 = known directory/author;
  Tier 4 = unknown/community → install-with-caution.
